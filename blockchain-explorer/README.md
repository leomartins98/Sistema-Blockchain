# Blockchain Explorer

## Overview
This project is a simple blockchain explorer built with React, Vite, and TailwindCSS. It allows users to interact with a mini-blockchain, view blocks, pending transactions, and create new transactions.

## Features
- View a list of blocks in the blockchain.
- View detailed information about each block.
- Display pending transactions in the mempool.
- Create new transactions through a user-friendly form.
- Mine blocks and validate the blockchain.

## Technologies Used
- **React**: A JavaScript library for building user interfaces.
- **Vite**: A fast build tool and development server for modern web projects.
- **TailwindCSS**: A utility-first CSS framework for styling.
- **Axios**: A promise-based HTTP client for making requests to the blockchain API.

## Setup Instructions

1. **Clone the repository**:
   ```bash
   git clone <repository-url>
   cd blockchain-explorer
   ```

2. **Install dependencies**:
   ```bash
   npm install
   ```

3. **Run the development server**:
   ```bash
   npm run dev
   ```

4. **Open your browser**:
   Navigate to `http://localhost:3000` to view the application.

## Usage
- Use the navigation to view the list of blocks and pending transactions.
- Fill out the transaction form to create a new transaction and click "Mine Block" to process it.
- Check the validity of the blockchain using the provided functionality.
